API

JSON endpoints for programmatic access to pipeline results and statistics.

GET /api/ This endpoint index
GET /api/tournament Tournament leaderboard data
GET /api/manifest Master manifest with per-group metadata
GET /api/regime Regime series, distribution, current state
GET /api/portfolio Portfolio summary (avg Sharpe, accuracy, trades)
GET /api/equity-curves List equity curve image paths
GET /api/comparison LSTM vs MTL comparison data
GET /api/sweep-curves Sweep experiment equity curve paths
GET /api/health Health check
GET /files/<path> Serve files from results/ or models/ (path-traversal safe)

Usage

curl http://localhost:5000/api/portfolio
curl http://localhost:5000/api/regime
curl http://localhost:5000/api/tournament
curl http://localhost:5000/files/results/equity_curve_portfolio.png
